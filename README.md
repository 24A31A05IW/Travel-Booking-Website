# ✈️ Travel Booking Website

A full-stack Travel Booking Web Application that allows users to search flights, explore destinations, and manage bookings through a clean and interactive interface.

---

## 🌍 Project Overview

This project is designed to simplify travel planning by combining a responsive frontend with a powerful backend API. Users can browse available flights, authenticate securely, and manage their bookings efficiently.

---

## 🚀 Features

- 🔍 Search and explore flights  
- 🧾 Book and manage travel reservations  
- 🔐 User authentication using JWT  
- 🛠 Admin routes for management  
- 📱 Responsive and clean UI  

---

## 🛠 Tech Stack

### Frontend
- HTML  
- CSS  
- JavaScript  

### Backend
- Python (Flask)  
- REST APIs  
- JWT Authentication

## ⚙️ Setup Instructions

### 🔧 Backend Setup

```bash
cd backend
pip install -r requirements.txt
python app.py
```

---

### 🌐 Frontend Setup

Simply open:

```
frontend/index.html
```

in your browser.
