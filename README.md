\# ✈️ Travel Booking Website



A full-stack travel booking web application that allows users to search flights, book tickets, and manage bookings.



\## 🚀 Features

\- Flight search and booking

\- User authentication (JWT)

\- Admin routes

\- Responsive frontend UI



\## 🛠 Tech Stack



\*\*Frontend:\*\*

\- HTML, CSS, JavaScript



\*\*Backend:\*\*

\- Python (Flask)



\## 📁 Project Structure

```

frontend/ → UI (HTML, CSS, JS)

backend/ → Server \& APIs

```



\## ⚙️ Setup Instructions



\### Backend

```

cd backend

pip install -r requirements.txt

python app.py

```



\### Frontend

Open `frontend/index.html` in browser
